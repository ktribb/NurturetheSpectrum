=== DATABASE SCHEMA ===
Table: listings
- id (auto-increment primary key)
- name (text, required)
- slug (text, unique — auto-generated from name)
- type (enum: Agency, Individual, Platform)
- city (text)
- county (enum: Philadelphia, Delaware County, Bucks County, Chester County)
- specializations (text array or JSON)
- certifications (text array or JSON)
- website (text)
- email (text)
- phone (text)
- hourly_rate (text)
- years_experience (integer, nullable)
- description (text)
- logo_url (text, nullable)
- tier (enum: Free, Featured, Verified) default Free
- status (enum: Published, Pending, Inactive) default Pending
- created_at (timestamp)
- updated_at (timestamp)
